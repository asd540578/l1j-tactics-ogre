仕様等

  • 始めに
    • この鯖でやることは雑魚とボスと人をひたすら狩るだけです。
    面倒なクエスト等は一切用意していません。

  • お金
    • このサーバーでは、ゴートという単位が存在します。
    1ゴートは100ｋアデナに相当します。
    敵からは、アデナではなくゴートをドロップします。

  • 対人関連
    •  [Ｋ／Ｄ]
    Ｋ／Ｄを個別にカウントします。

    [死亡時　その１]
    ＭＯＢで死亡すると2.5%ロストします。
    ＰＣで死亡すると5%ロストします。
    また、ＰＣに殺されるとＰＫログが流れます。

    [死亡時　その２]
    ＰＫでドロップが発生した場合は、ＬＡを取った人がアイテムを取得します。
    アイテム取得時は、ＭＯＢドロップと同様にメッセージが表示されます。

    [バックアタック]
    対象をバックアタック（斜め後ろ含む）した場合は、
    一定の攻撃成功値及び追加打撃値が加わります。
    弓は対象外です。

  • ＰＴ関連
    •  ＰＴリーダー以外でも招待が可能です。
  • 武器について
    •  ほとんどの武器が、火力と攻撃成功率が反比例します。

  • 武器属性について
    •  火、水、地、風、光、闇の6種類があります。
    また、5段階の強化が可能です。

    段階火水風地光闇
    1火の水の風の地の光の闇の
    2紅蓮の水鏡の疾風の砂塵の光輝の暗黒の
    3煉獄の氷河の烈空の地裂の月光の常闇の
    4灼熱の氷海の竜巻の烈震の閃光の漆黒の
    5劫火の氷土の風刃の激震の煌めきの深淵の

    ダメージは固定ではなく、下記表の通り計算を行います。
    また、光・闇の属性は火・水・風・地の属性を基にレジストの計算を行います。

  • クラス別コメント
    •  変更点を簡単に記載します。
    基準と記載するものは、レベルから70引いた数字を基に、
    何かしら計算するものと考えて下さい。
    デバフ系は軒並み効果アップ継続時間ダウンで
    バフ系は軒並み効果アップ継続時間アップしています。

    [ナイト]
    片手の火力が上がってるので、
    スタン＋片手でかなりＫＩＬＬが取りやすいと思います。
    ＣＢ無敵ゲーになるとつまらないので、ＣＢの確率を下げました。

    ＣＢ→確率低下
    スタン→秒数アップ
    リダク→リダク値アップ＋基準

    [ダークエルフ]
    タイマン最強クラス。（多分）
    ＣＢがあってもナイトを吹っ飛ばせます。

    [ドラゴンナイト]
    槍や斧をかなり強化したので、是非使って欲しいクラス。。。
    火力的にはＤＥにも劣りません。
    魔法はレベルよるものが多いので、極めれば強くなると思います。

    ドラゴンスキン→リダク値アップ＋基準
    ガードブレイク→AC低下アップ＋基準
    モータルボディ→基礎値＋基準
    ホラーオブデス→基礎値＋基準

    [ウィズ]
    ＨＷ→128秒
    高ＳＰ装備が多いので、多分十分強いです。

    [エルフ]
    盾と片手弓を充実させたので、両手弓以外も使ってみて下さい。
    スキン→基礎値＋基準
    ポルート・バインド・イレース・ゲイル→時間低下

    [イリュ] イリュージョン系魔法の大幅強化してます。
    この鯖の武器の特性上、ＡＣが物を言うのでダイアゴレが重宝すると思います。
    オーガ→基礎値＋基準
    ダイア→基礎値＋基準
    アバター→基礎値＋基準

開放コマンド

  • .desc
    • 自分のステータスを確認出来ます。

  • .descnpc ＮＰＣ名称
    • ＮＰＣのステータスを確認出来ます。
    例）.descnpc 彩雲のオルバ

  • .potlog on/off
    • ＰＯＴＬＯＧの表示非表示を変更します。
    例）.potlog off

  • .droplog on/off
    • ＤＲＯＰＬＯＧの表示非表示を変更します。
    例）.droplog off

  • .addskill
    • 全スキルを習得します。

  • .loc
    • 自分の位置を確認できます。
