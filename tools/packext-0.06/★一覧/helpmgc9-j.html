ヘルプ：ウィザードの魔法レベル9


ライトニングストーム

種類：攻撃、風

性向：ニュートラル

消耗MP/HP：48/0

持続時間：一瞬

対象：ターゲットの3セル以内の敵

効果：強力な雷を落とし、対象と対象の周囲にいる全ての敵にダメージを与えます。

フォグオブスリーピング

種類：呪い、水

性向：カオティック

消耗MP/HP：40/0

材料：魔力の石1個

持続時間：32秒

対象：術者から10セル以内の対象

効果：対象を中心とした周囲に霧を発生させ、範囲内にいるPCやNPCを一定の確率で眠らせます。眠ったPCやNPCは攻撃を受けると目覚めます。アライメント-5

イミューントゥハーム

種類：変化

性向：ロウフル

消耗MP/HP：30/0

材料：魔力の石2個

持続時間：32秒

対象：術者または術者から15セル以内の血盟員1人

効果：対象に強力なマジックバリアを張り、受けるダメージを半分に減らします。

マステレポート

種類：操作

性向：ニュートラル

消耗MP/HP：60/0

材料：魔力の石1個

持続時間：一瞬

対象：術者と、その周囲の2セル内にいる術者が属する血盟の血盟員と、術者が連れているモンスターたち

効果：全員をテレポートさせます。テレポート地点の制限は、テレポート魔法と同じです。

ファイアーストーム

種類：攻撃、火

性向：ニュートラル

消耗MP/HP：48/0

持続時間：一瞬

対象：術者の周囲3セル内にいる全ての敵

効果：術者の周辺一定範囲に炎の爆風を起こします。

ディケイポーション

種類：操作

性向：ニュートラル

消耗MP/HP：20/0

材料：魔力の石2個

持続時間：16秒

対象：術者から3セル以内のPC1人またはNPC1人

効果：対象はポーション状のいかなるアイテムも使用できなくなります。

カウンターディテクション

種類：操作

性向：ニュートラル

消耗MP/HP：30/0

持続時間：一瞬

対象：術者から15セル以内の対象

効果：術者から15セル以内の透明な敵にダメージを与えて、透明状態を解除します。
