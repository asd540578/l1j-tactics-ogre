Tactics Ogre Serverについて

＜超重要なお知らせ＞
2012/7/1（日）の23:59を持ってサーバーを（一旦）閉鎖します。
理由は「ＧＭの作業速度の低下」及び「人口の減少」です。
短い間でしたが、ここまで遊んで頂きありがとうございました。
現在はAlaska鯖が盛況なようですので、是非行ってみて下さい。
（L1Jユーザが減らないと嬉しいです）

もし当サーバーを再稼働することがあれば、追加要素を加えた上でワイプ（※）して再開したいと思います。
※現在のレベル・装備等考慮して初期特典をつけます。

＜重要なお知らせ＞
Ｓ装備クエスト始めました。

タクティクスオウガの要素を取込んだPKサーバーを作成しようとしてます。
中々時間が取れないため、不定期になるかと思いますが楽しく遊んで頂ければ幸いです。
アカウント作成の際は、本鯖・他エミュ鯖と異なるアカパスの使用をお願いします。

サーバーは24h稼働します。
手動以外のシャットダウンは、全て20分前からカウントします。

クランのエンブレムについて

    クランのエンブレムが、表示されないという問題がありました。
    対処としては、プリを操作する方が自分のリネフォルダにある
    Emblem****.dat（*は数字）のファイルを全て削除してから、
    エンブレムをアップすることで正常にサーバーに保存されるようです。
