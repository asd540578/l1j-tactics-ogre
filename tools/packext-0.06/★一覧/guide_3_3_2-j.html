ウィザード：
ギランの町の北西の出口から出て、ギラン領地を通り過ぎて11時の方角にまっすぐ行くと川があります。そこにかかった橋を渡れば、ドラゴンバレーです。
そこはアンタラスの棲処につながるドラゴンバレーのダンジョンともつながってるんですけど、スケルトンマークスマン、スケルトンファイター、スケルトンガード、スパルトイとかのアンデッドが出るんですよ。その他にも、ハーピー、スコーピオン、オーガ、コカトリス、ドレイクなどのモンスターも出ますよ。
オーガ、ハーピー、スコーピオン以外のほとんどのやつが先制攻撃をかけてくるから気をつけてくださいね。ドラゴンバレーの奥深くまで入ってしまうと、ドレイクが群れだって現れることもあるし、ブラック エルダーもいるから充分に気をつけてください。では、幸運を祈ります。
